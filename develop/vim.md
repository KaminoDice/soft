剪切板
1. vim 有12个剪切板, 分别是 `0-9,a,",+`, 其中 `y/d/x` 等指令将内容复制到 `"` 的剪切板中, 使用 `p` 也是复制 '"' 剪切板的内容.
2. 使用 `:reg` 可以查看所有剪切板的内容
3. 复制到指定剪切板: 在要使用的剪切板前加 `"` 即可. 如复制到`+`剪切板: `"+y`.
4. `+` 剪切板是系统剪切板.
